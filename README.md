# RKF45 ODE Solver – Project 2

## Description
This project solves a first-order ordinary differential equation using the
Runge–Kutta–Fehlberg (RKF45) method and compares it with Euler’s method.

The ODE solved is:
y' = -y + ln(x)

## Requirements
- Python 3.x
- NumPy
- Matplotlib
